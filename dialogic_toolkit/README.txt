Welcome to your QAT-Kit.

Copy/paste the text from "Project_Instructions.txt" into a new ChatGPT Project under the "Instructions" field. 
Copy all other files to "Project Files" within the same Project.

To use:
Move an existing thread or create a new one in the Project.
Prompt ChatGPT to list available tools.
- Ex: “What tools are available in this QAT-Kit?”
Invoke a tool on the whole thread, parts of the thread, or entirely new options. 
- Ex: “Please run StressTest on this idea: ______”

To extend:
Add new entries to the toolkit and qat_manifest JSON  files.
